tinyC, compiled to wasm, running in your browser

This page loads tinyc.wasm — the entire tinyC compiler + in-tree wasm linker, both built from this repository as a single wasm32-wasi module. It compiles a tinyC source to a wasm32 object, links it against a pre-built runtime, then runs the produced wasm — all in JavaScript, with no server-side compilation.
